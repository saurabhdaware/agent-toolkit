## `agent-browser` Installation

Steps to install `agent-browser` CLI

- Step 1: Get confirmation from user if they want to install it using AskQuestion tool
- Step 2: If user approves, install it using following commands-
  ```sh
  npm install -g agent-browser
  agent-browser install  # Download Chromium
  ```
- Step 3: Check if the CLI is installed as expected
  ```sh
  agent-browser --help
  ```
